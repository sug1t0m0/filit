# filit

CLI that bundles selected files into a single browser page so tab-aware LLMs (Gemini in Chrome, etc.) can ingest them at once.

Inspired by [difit](https://github.com/yoshiko-pg/difit) — but instead of reviewing a diff, filit lets you pick arbitrary files, preview the assembled context in your browser, and hand the whole tab to an LLM.

## Status

Work in progress. Not yet usable.
